What you get

GitJoker-C2 provides operational governance: validation, coordination and authorization before execution. All levels enforce the same minimal chain:

Event → IPR/HRR → DGV → Certified Time → Evidence Bundle
Fail-closed rule: if a required element is missing or unverifiable, execution is blocked.

Individuals / Professionals

Levels are additive: higher levels inherit the minimal chain and add stricter governance controls.
